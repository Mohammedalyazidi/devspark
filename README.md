#  React + TypeScript + Vite Project

[![TypeScript](https://img.shields.io/badge/TypeScript-5.x-blue)](https://www.typescriptlang.org/)

A **modern React project template** using **TypeScript** and **Vite**, optimized for performance and developer experience. Includes ESLint, optional React Compiler, and Fast Refresh using Babel or SWC.

---

##  Features

- **React 18 + TypeScript**  
- **Vite** for blazing fast development and optimized builds  
- Fast Refresh via:
  - [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) (Babel or [oxc](https://oxc.rs/) when used in [rollup-vite](https://vite.dev/guide/rolldown))  
  - [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react-swc) (uses [SWC](https://swc.rs/))  
- ESLint with type-aware linting rules  
- Optional **React Compiler** support  
- Well-structured project folder for scalable applications
